Your Website Content

This is an example of how the Incorta Chatbot can be embedded into your website. The chatbot is fully responsive and will adapt to different container sizes.

Embedding Options

  • Sidebar Integration: As shown on the right, perfect for documentation sites or help centers
  • Modal/Popup: Click the blue button in the bottom right to see a floating chat window
  • Full Page: Can be embedded as a full-page experience
  • Inline: Embedded within page content at any size

Features

  • Fully responsive design that adapts to container size
  • Dark mode support
  • Streaming responses with typing animation
  • Source citations and related questions
  • Mobile-optimized interface
  • No fixed viewport dependencies - works in any container

Implementation

Simply embed the chatbot using an iframe or integrate it directly into your React/Next.js application:

<iframe 
    src="http://localhost:3000" 
    width="100%" 
    height="600px"
    style="border: none; border-radius: 8px;"
></iframe>
